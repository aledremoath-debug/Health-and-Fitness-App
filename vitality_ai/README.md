# VitalityAI - تطبيق الصحة واللياقة البدنية الذكي 🚀

**VitalityAI** هو تطبيق متطور للهواتف المحمولة وسطح المكتب، تم بناؤه باستخدام **Flutter**، ويهدف إلى مساعدة المستخدمين على تتبع صحتهم، لياقتهم البدنية، وأهدافهم الغذائية باستخدام واجهات عصرية وتجربة مستخدم متميزة.

---

## ✨ المميزات الرئيسية | Main Features

-   **📊 حسابات صحية دقيقة**:
    -   حساب مؤشر كتلة الجسم (BMI).
    -   حساب معدل الأيض الأساسي (BMR).
    -   حساب إجمالي نفقات الطاقة اليومية (TDEE).
    -   تحديد السعرات الحرارية المستهدفة بناءً على الأهداف (فقدان وزن، ثبات، زيادة عضلات).
    -   توزيع الماكروز (Macros) حسب نوع الحمية.
    -   حساب كمية المياه الموصى بها يومياً.

-   **👤 متخيل الجسم (Body Visualizer)**: عرض مرئي وشامل لحالة الجسم بناءً على القياسات المدخلة.
-   **🥗 خطط الوجبات والتمارين**: قوائم منظمة للوجبات والتمارين الرياضية المقترحة.
-   **📈 تتبع التقدم**: رسوم بيانية تفاعلية وتفصيلية لمراقبة التطور بمرور الوقت.
-   **📅 تحديات ومقالات**: قسم خاص للتحديات الرياضية ومقالات تعليمية لتحسين نمط الحياة.
-   **🌓 دعم الوضع المظلم (Dark Mode)**: واجهات مريحة للعين تدعم الثيم الفاتح والمظلم.
-   **🌍 ثنائي اللغة**: يدعم اللغة العربية والإنجليزية بشكل كامل.
-   **📱 تصميم متجاوب (Responsive)**: تجربة مستخدم سلسة على الهواتف، الأجهزة اللوحية، وأجهزة الكمبيوتر.

---

## 🛠 التقنيات المستخدمة | Tech Stack

-   **Framework**: [Flutter](https://flutter.dev/) (SDK ^3.8.1)
-   **State Management**: [Provider](https://pub.dev/packages/provider)
-   **Data Visualization**: [fl_chart](https://pub.dev/packages/fl_chart)
-   **Typography**: [Google Fonts](https://pub.dev/packages/google_fonts)
-   **Architecture**: Clean & Modular Structure (Models, Providers, Screens, Widgets).

## 🚀 كيفية التشغيل | Getting Started

### المتطلبات المسبقة:
-   تثبيت Flutter SDK.
-   تثبيت Android Studio أو VS Code مع إضافات Flutter.

### خطوات التشغيل:

1. قم بتحميل المشروع (Clone the repository).
2. افتح المجلد في محرر الأكواد الخاص بك.
3. قم بتنزيل الحزم المطلوبة:
   ```bash
   flutter pub get
   ```
4. قم بتشغيل التطبيق:
   ```bash
   flutter run
   ```
